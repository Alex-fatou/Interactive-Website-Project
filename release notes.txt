Next Steps & Guides
The project is built using HTML, CSS, JavaScript, and JSON.

index.html serves as the main page of the site.

style.css contains the global styles and general classes used throughout the project.

Each page is located in the root folder, with its corresponding CSS and JavaScript files organized in separate folders. The folder names match the page names. For example, the "About" page consists of about.html, about.css, and about.js.

All images are stored in the images folder. However, SVG files for the homepage and project sections are kept separately.

The json folder contains the filter.json file, which manages the filtering functionality.

Moving forward, I believe the CSS and JavaScript code structure could be improved for better organization and maintainability.
